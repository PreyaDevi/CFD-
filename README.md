# CFD Analysis of Supersonic Flow Over a Wedge

##  Objective

To analyze supersonic flow over a wedge using analytical methods and ANSYS CFD simulation.

## Tools Used

* ANSYS Workbench
* k–ω SST Turbulence Model
* Excel / MATLAB

## Problem Description

This project studies a 2D supersonic flow over a wedge with a deflection angle of 40°.
Mach numbers analyzed:

* 1.75
* 2.25
* 2.75
* 3.25

## Methodology

* Analytical solution using θ–β–M relation
* Numerical simulation using ANSYS
* Comparison of results

##  Results

* Shock waves observed in all cases
* Increase in pressure and temperature across shock
* CFD results validated analytical behavior
